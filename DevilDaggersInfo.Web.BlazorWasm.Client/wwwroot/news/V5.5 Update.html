Global Custom Leaderboards

Added global custom leaderboards, there is one global leaderboard for every leaderboard category. The global ranking is based on the ranks and daggers achieved by the player in all the leaderboards for that category. It's still a little bit of a work in progress, but the idea is there.

Custom Leaderboard Score API V2

  • Rewritten API with improved performance for score submissions.
  • Replay files are now validated upon submit.
  • Fixed saving incorrect homing stored value upon submit.
  • Fixed replay submissions being able to override highscore by a very small amount.

Replay Parsing

  • Implemented new library that is able to parse (local) replay files, as well as build replay files from scratch. These features can be used for analyzing gameplay, or possibly for writing a TAS for a certain spawnset.
  • The parsing feature can be tried out when viewing a custom leaderboard score. It parses the replay and displays all individual events on the page. See here for an example.
  • Most of the events have been figured out, but not all is known yet (unknown values are named A, B, C, etc).
  • Maybe I'll make a new tool called DevilDaggersReplayEditor some time in the future, who knows...

Open Source

The entire project is now open source!

If devildaggers.info is useful to you, please consider giving it a star on GitHub. This lets me know the work is appreciated. ❤️

Open sourcing the project was done for a couple reasons, the most important one being that I just want to share this project with everyone else. I'm quite happy with how it turned out, and don't want it to be lost when I stop working on it.

Another reason is that the two editors are also open source for their own reasons, and I plan to eventually rewrite them to be improved and cross-platform. The new editors would live in this Git repository to keep everything in one place, and to make it easier for them to be embedded on the website. It wouldn't make sense for the source code for the new editors to not be publicly available, so everything is now open to the public.

That being said, I will likely be starting the development for the new editors soon. Will send updates on Discord.
