# Death types

There are 17 death types in the latest version of the game. These are used throughout multiple formats, sometimes as 8-bit integers, sometimes as 32-bit integers.

| Death type  | Value |
|-------------|-------|
| FALLEN      | 0     |
| SWARMED     | 1     |
| IMPALED     | 2     |
| GORED       | 3     |
| INFESTED    | 4     |
| OPENED      | 5     |
| PURGED      | 6     |
| DESECRATED  | 7     |
| SACRIFICED  | 8     |
| EVISCERATED | 9     |
| ANNIHILATED | 10    |
| INTOXICATED | 11    |
| ENVENOMATED | 12    |
| INCARNATED  | 13    |
| DISCARNATED | 14    |
| ENTANGLED   | 15    |
| HAUNTED     | 16    |
